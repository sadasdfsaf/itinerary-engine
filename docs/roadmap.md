# Roadmap

## v0.1

- Stable request / itinerary schemas
- Static catalog adapter
- Simple candidate selector
- Baseline planner
- Rule-based edit parser
- Patch engine with day-level partial replan
- FastAPI server
- Next.js playground
- Example payloads and architecture docs

## v0.2

- Constraint bundles: weather, opening hours, dietary filters
- Better ranking signals and diversity controls
- Travel-time-aware day shaping
- JSON regression fixtures
- More destinations and adapter examples

## v0.3

- Benchmark suite for generation and edit quality
- Provider comparison harness
- Versioned schema migration strategy
- Optional LLM-backed parser and planner adapters

## v0.4

- Stronger edit operations: swap, compress, expand, reorder blocks
- Map and routing adapter contracts
- Batch evaluation CLI
- Embeddable SDK examples for server and frontend apps
