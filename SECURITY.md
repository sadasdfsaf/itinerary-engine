# Security Policy

## Supported Versions

This repository is pre-1.0. Security fixes will target the latest main branch first.

## Reporting a Vulnerability

Please do not open a public issue for a suspected vulnerability.

Instead, send:

- a short summary
- affected component
- reproduction steps
- expected impact

to the maintainer contact you publish for this repository, or use GitHub Security Advisories once the repository is public.

## Scope

Relevant issues include:

- remote code execution
- unsafe deserialization
- secret leakage
- SSRF or credential forwarding in adapters
- insecure default exposure in API examples
